Bases del concurso 2014

Registro y participación

  • Para participar en el concurso, una vez te has registrado debes subir una fotografía.

  • Dicha fotografía debe ser geolocalizada con una etiqueta para poder identificar el lugar en el que fue tomada.


Proceso de votación

  • Para poder votar las fotografías, es condición obligatoria haber subido previamente una.

  • Cuando se vaya a hacer la votación, se deberán seguir unas instrucciones para crear un certificado único para cada participante.

  • De esta forma, se garantiza que la votación es anónima, y posteriormente se pueden comprobar los votos de la urna sin ningún problema.
